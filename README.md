# ft_gkrellm

### Description

*The goal of this project is to recode GKrellM. Ha-ha, what a jokeðŸ˜…*

### Requirements
- macOS
- installed ncurses
- installed SDL

How to start the program:
```
make
./ft_gkrellm -g || sdl || ncurses

Flags:
-h: to show list of flags
sdl: to launch graphical UI
ncurses: to launch text UI
```

Some usefull buttons:
```
In SDL:
ESQ - exit

In Ncurses:
ESQ - exit
UP and DOWN - moving
```
